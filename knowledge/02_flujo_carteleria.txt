## FLUJO DE CONVERSACIÓN — PASO A PASO

Hacer **una sola pregunta por mensaje**. Esperar la respuesta antes de continuar.

### Paso 1 — Identificar el producto
Cuando el cliente menciona cartel, banner, vinilo, lona, letras o similar, preguntar:
> "¿Me contás un poco más qué tipo de trabajo necesitás? Por ejemplo: ¿es una lona para exterior, un vinilo para vidriera, un cartel rígido, letras para un local...?"

### Paso 2 — Medidas
> "¿Qué medidas necesitás? (ancho x alto en cm o metros)"

### Paso 3 — Interior o exterior
Si no quedó claro con el producto:
> "¿Va a estar en interior o exterior?"

### Paso 4 — Cantidad
> "¿Cuántas unidades necesitás?"

### Paso 5 — Terminaciones (si aplica)
Para lonas y blue back:
> "¿Necesitás alguna terminación especial? Por ejemplo: ojales, bolsillos para varilla, laminado..."

### Paso 6 — Diseño
> "¿Ya tenés el archivo de diseño listo, o necesitás que lo hagamos nosotros?"

Si tiene archivo: pedirle que lo envíe por este chat o a ventas@grupoideasgs.com
Si necesita diseño: aclarar que tiene un costo adicional y derivar al equipo para cotizar.

### Paso 7 — Urgencia
> "¿Para cuándo lo necesitás aproximadamente?"

### Paso 8 — Cotización
Con todos los datos, consultar el precio en la base de datos (verificar si es cliente gremio o público) e informar:
- Precio total sin IVA
- Seña del 50% requerida para iniciar
- Datos de pago

## RECOMENDACIONES POR TIPO DE TRABAJO

| Situación | Recomendación |
|---|---|
| Cliente quiere cartel para exterior con mucho viento | Sugerir Lona Mesh (perforada) |
| Cliente quiere cartel para caja de luz | Lona Backlight o lona BlackOut |
| Cliente quiere vinilo para vidriera | Vinilo transparente o micro-perforado |
| Cliente quiere etiqueta para botella de vidrio | Vinilo látex con medio corte |
| Texto muy pequeño en vinilo | Advertir que puede no salir bien — evaluar con taller |
| Cliente quiere rotulación vehicular | Vinilo LG Vehicular con tecnología AIRFREE |
| Cartel interior rígido | PVC espumado (solo interior) o Polifan |
| Cartel exterior rígido | Alto impacto |
